# Installation

Install MDC Syntax using your preferred package manager:

::code-group

```bash [npm]
npm install mdc-syntax
```

```bash [pnpm]
pnpm add mdc-syntax
```

```bash [yarn]
yarn add mdc-syntax
```

```bash [bun]
bun add mdc-syntax
```

::

## Simple usage


```vue
<script setup lang="ts">
import { MDC } from 'mdc-syntax/vue'

const content = '# Hello World\n\nThis is a **test**.'
</script>

<template>
  <MDC :value="content" />
</template>
```
